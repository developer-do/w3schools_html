HTML JavaScript

JavaScript는 HTML 페이지를 더욱 역동적이고 대화식으로 만듭니다.

HTML < /script > 태그

이 < / script > 태그는 클라이언트 측 스크립트 (javascript)를 정의하는데 사용됩니다.

< / script > 요소는 스크립트 문을 포함하거나 src 특성을 통해 외부 스크립트 파일을 가리킵니다.

자바스크립트의 일반적인 용도는 이미지 조작, 양식 유효성 검사 및 콘텐츠의 동적 변경입니다.

HTML 요소를 선택하기 위해 JavaScript는 자주 document.getElementById() 메소드를 사용합니다.

이 JavaScript 예제는 id = "demo"인 HTML 요소에 "Hello JavaScript!" 라고 추가합니다.

HTML < noscript >태그

이 < noscript > 태그는 브라우저에서 스크립트를 비활성 했거나 클라이언트 측 스크립트를 지원하지 않는 브라우저를 가진 사용자를 위해 대체 컨텐츠를 제공하는데 사용됩니다.
