HTML The class Attribute

class 속성 사용하기

HTML class 속성은 동일한 클래스 이름을 가진 요소에 대한 동일한 스타일을 정의하는데 사용됩니다.

따라서 동일한 class 속성을 가진 모든 HTML 요소는 동일한 형식 및 스타일을 갖습니다.

여기 < div > 에는 동일한 클래스 이름을 가리키는 세 가지 요소가 있습니다.

ex)

London

London is the capital of England.

Paris

Paris is the capital of France.

Tokyo

Tokyo is the capital of Japan.

인라인 요소에서 class 속성 사용

class 인라인 요소에서 HTML 속성을 사용할 수도 있습니다.

My Important Heading

This is some important text.

Tip: class 속성은 모든 HTML 요소에서 사용할 수 있습니다.

Note: 클래스 이름은 대소 문자를 구분합니다!

여러 클래스

HTML 요소는 클래스 이름을 두 개 이상 가질 수 있으며 각 클래스 이름은 공백으로 구분해야 합니다.

ex)

클래스 이름이 "city"인 스타일 요소, 클래스 이름이 "main"인 요소 스타일

London

Paris

Tokyo

위 예제에서 첫 번째 < h2 >요소는 "city"클래스와 "main"클래스에 속합니다.

다른 태그는 같은 클래스를 공유 할 수 있습니다.

< h2 > 및 < p >와 같은 다른 태그는 동일한 클래스 이름을 가질 수 있으므로 동일한 스타일을 공유합니다.

ex)

Paris

Paris is the capital of France

Javascript의 class 속성 사용

클래스 이름은 JavaScript가 지정된 클래스 이름을 가진 요소에 대한 특정 작업을 수행하는데 사용 될 수도 있습니다.

JavaScript는 다음 getElementsByClassName()메소드를 사용하여 지정된 클래스 이름을 가진 요소에 엑세스 할 수 있습니다.

ex)

사용자가 버튼을 클릭하면 클래스 이름이 "city"인 모든 요소를 숨깁니다.
