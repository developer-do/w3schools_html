HTML Iframe

iframe은 웹 페이지 내에 웹 페이지를 표시하는 데 사용됩니다.

iframe 구문

HTML iframe은 다음 < iframe > 태그로 정의됩니다.

ex)

< ifrmae src="URL" > < /iframe >

src 속성은 인라인 프레임 페이지의 URL (웹 주소)을 지정합니다.

iframe - 높이 및 너비 설정

높이 및 너비 특성을 사용하여 iframe의 크기를 지정합니다.

속성값은 기본적으로 픽셀 단위로 지정되지만 백분율 (예: "80%") 일수도 있습니다.

ex)

< iframe src="../17. HTML Tables/index.html" frameborder="0" style="border: 2px solid #ff6d24;" height="200" width="300">< /iframe > 아래와 같다.

또는 CSS를 사용하여 iframe의 높이와 너비를 설정할 수 있습니다.

ex)

< iframe src="../18. HTML Lists/index.html" frameborder="0" style="width: 100%; height: 500px; border: 2px solid #ff6d24;">< /iframe > 아래와 같다.

iframe - 테두리 제거

기본적으로 iframe에는 테두리가 있습니다.

테두리를 제거하려면 style 속성을 추가하고 CSS border속성을 사용하십시오.

또는 iframe 속성을 frameborder="0"으로 맞추십시오.

ex)

< iframe src="../11. HTML Quotations/index.html" style="border:none;" frameborder="0">< /iframe > 아래와 같다.

CSS를 사용하면 iframe의 테두리 크기, 스타일 및 색상을 변경할 수도 있습니다.

ex)

< iframe src="../12. HTML Comments/index.html" style="border:2px solid #ff6d24;">< /iframe > 아래와 같다.

iframe - 링크 대상

iframe은 링크 대상 프레임으로 사용할 수 있습니다.

링크의 대상 속성은 iframe의 name 속성을 참조해야합니다.

< iframe src="" style="width: 100%; height: 500px;" name="iframe_a">< /iframe >

< p >< a href="../14. HTML CSS/index.html" target="iframe_a">14. HTML CSS< /a >< /p >
