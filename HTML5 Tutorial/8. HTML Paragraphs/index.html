HTML Paragraph

HTML 단락

HTML < p > 요소는 단락을 정의합니다.

    ex)
    < p >This is a paragraph.< /p >
    < p >This is another paragraph.< /p >

Note: 브라우저는 단락 앞뒤에 공백을 자동으로 추가합니다.(여백)

HTML 표시

HTML이 어떻게 표시 될지는 확신 할 수 없습니다.

크고 작은 화면 or 크기가 조정 된 창은 다른 결과를 만듭니다.

HTML을 사용하면 HTML 코드에 여분의 공백이나 여분의 줄을 추가하여 출력을 변경할 수 없습니다.

페이지가 표시 될 때 브라우저는 여분의 공백과 여분의 행을 제거합니다.

ex)

This paragraph contains a lot of lines in the source code, but the browser ignores it.

This paragraph contains a lot of spaces in the source code, but the browser ignores it. 

      < p >
        This paragraph
        contains a lot of lines
        in the source code,
        but the browser
        ignores it.
      < /p >
      < p >
        This paragraph
        contains            a lot of spaces
        in the source       code,
        but the           browser
        ignores it.
      < /p >

끝 태그를 잊지 마라.

대부분의 브라우저는 종료 태그를 잊어 버린 경우에도 HTML을 올바르게 표시합니다.

  < p >This is a paragraph.
  < p >This is another paragraph

위의 예제는 대부분의 브라우저에서 작동하지만, 의존하지는 않습니다.

위의 말을 설명하자면 브라우저들이 저것만 보고도 파싱은 해주지만, 저렇게 쓰면 작동을 안할수도 있다는 말과 동일하다.

Note: 종료 태그를 삭제하면 예기치 않은 결과 또는 오류가 발생할 수 있습니다.

The Poem Problem(시 문제)

아래의 시는 한 줄에 표시됩니다 :

ex)

My Bonnie lies over the ocean. My Bonnie lies over the sea. My Bonnie lies over the ocean. Oh, bring back my Bonnie to me.

HTML < pre > 요소

HTML < pre >요소는 서식이 미리 지정된 텍스트를 정의합니다.

< pre > 요소 내부의 텍스트는 고정 너비 글꼴 (일반적으로 Courier)로 표시되며 공백과 줄 바꿈을 모두 유지합니다.

ex)

    My Bonnie lies over the ocean.

    My Bonnie lies over the sea.
    
    My Bonnie lies over the ocean.
    
    Oh, bring back my Bonnie to me.
