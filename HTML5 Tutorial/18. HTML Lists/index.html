HTML Lists

정렬되지 않은 HTML 목록

정렬되지 않은 목록은 < ul > 태그로 시작합니다. 각 목록 항목은 < li > 태그로 시작합니다.

목록 항목은 기본적으로 글 머리 기호 (작은 검은 색 원)로 표시됩니다.

ex)

  • Coffee
  • Tea
  • Milk

정렬되지 않은 HTML 목록 - 목록 항목 마커 선택

CSS list-style-type 속성은 목록 항목 마커의 스타일을 정의하는 데 사용 됩니다.

Value Description
disc Sets the list item marker to a bullet (default)
circle Sets the list item marker to a circle
square Sets the list item marker to a aquare
none The list items will not be marked

ex) disc

  • Coffee
  • Tea
  • Milk
  • Coffee
  • Tea
  • Milk
  • Coffee
  • Tea
  • Milk
  • Coffee
  • Tea
  • Milk

순서가 지정된 HTML 목록

순서가 지정된 목록은 < ol > 태그로 시작합니다. 각 목록 항목은 < li >태그로 시작합니다.

목록 항목에는 기본적으로 숫자가 표시됩니다.

  1. Coffee
  2. Tea
  3. Milk

정렬된 HTML 목록 - 유형 속성

태그의 type 속성은 < ol > 목록 항목 마커의 유형을 정의합니다.

Value Description
type="1" The list items will be numbered with numbers (default)
type="A" The list items will be numbered with uppercase letters
type="a" The list items will be numbered with lowercase letters
type="I" The list items will be numbered with uppercase roman numbers
type="i" The list items will be numbered with lowercase roman numbers

ex) 번호

  1. Coffee
  2. Tea
  3. Milk

ex) 대문자

  1. Coffee
  2. Tea
  3. Milk

ex) 소문자

  1. Coffee
  2. Tea
  3. Milk

ex) 대문자 로마 숫자

  1. Coffee
  2. Tea
  3. Milk

ex) 소문자 로마 숫자

  1. Coffee
  2. Tea
  3. Milk

HTML 설명 목록

HTML은 설명 목록도 지원합니다.

설명 목록은 각 용어에 대한 설명과 함께 용어 목록입니다.

< dl > 태그는 상기 설명 목록을 정의 < dt > 태그 용어(이름)를 정의하고, < dd >태그는 각각의 용어 설명

ex)

Coffee
-black hot drink
Milk
- white cold drink

중첩 된 HTML 목록

목록을 중첩 할 수 있습니다. (목록 내부에 나열)

ex)

  • Coffee
  • Tea
    • Black tea
    • Green tea
  • Milk

Tip: 목록 항목에는 새 목록과 이미지 및 링크 같은 기타 HTML 요소가 포함 될 수 있습니다.

제어 목록 계산

기본적으로 순서가 지정된 목록의 수는 1부터 시작됩니다. 지정한 수에서 계산을 시작하려면 다음 start 특성을 사용할 수 있습니다.

ex)

  1. Coffee
  2. Tea
  3. Milk

CSS가 있는 가로 목록

HTML 목록은 CSS를 사용하여 다양한 방식으로 스타일을 지정할 수 있습니다.

한 가지 일반적인 방법은 목록을 가로 방향으로 스타일을 지정하여 탐색 메뉴를 만드는 것입니다.

ex)

  • Home
  • News
  • Contact
  • About

단원 요약

  1. 정렬되지 않은 목록을 정의 하려면 HTML 에 < ul >태그를 사용하십시오.
  2. CSS list-style-type 속성을 사용하여 목록 항목 마커를 정의하십시오.
  3. HTML < ol > 요소를 사용하여 정렬된 목록을 정의하십시오.
  4. 번호 매기기 유형을 정의하려면 HTML 의 type 속성을 사용하십시오.
  5. HTML < li > 요소를 사용하여 목록 항목 정의
  6. HTML < dl > 요소를 사용하여 설명 목록 정의
  7. HTML < dt > 요소를 사용하여 설명 용어 정의
  8. HTML < dd > 요소를 사용하여 설명 목록에 용어를 설명합니다.
  9. 목록은 목록 내부에 중첩 될 수 있습니다.
  10. 목록 항목에는 다른 HTML 요소가 포함 될 수 있습니다.
  11. CSS 속성 float:left 하거나 display: inline 을 사용하여 가로로 목록을 배치 하십시오.
