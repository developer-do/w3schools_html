    ! 빈 HTML 요소. 내용이 없는 HTML 요소를 빈 요소라고한다. (empty element)
      < br > 같은 경우가 닫는 태그가 없는 빈 요소 라고한다.
      HTML5 에서는 빈 요소를 닫을 필요가 없지만, 엄격한 유효성 검사 or XML 파서로 문서를 읽을 수 있도록 해야하는 경우
       모든 HTML 요소를 올바르게 닫아야한다.
    ! 소문자 태그 사용
      HTML 태그는 대소 문자를 구분하지 않는다.  < P >와 < p >는 같다.
      하지만 W3C는 HTML에서 소문자를 권장하고,
      XHTML과 같이 더 엄격한 문서 유형의 경우 소문자를 요구한다.
