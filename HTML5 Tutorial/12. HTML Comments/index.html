HTML Comments (HTML 주석)

주석 태그는 HTML 소스 코드에 주석을 삽입하는데 사용됩니다.

HTML Comment Tags

다음 구문을 사용하여 HTML 소스에 주석을 추가 할 수 있습니다.

열기 태그에는 느낌표 (!)가 있지만 닫기 태그에는 느낌표가 없습니다.

Tip: 브라우저는 주석을 표시하지 않지만 HTML 소스 코드를 문서화하는 데 도움을 줄 수 있습니다.

의견을 통해 HTML에 알림 및 미리 알림을 추가 할 수 있습니다.

ex)

This is a paragraph.
