HTML id 속성

id 속성 사용하기

이 id 속성은 HTML 요소의 고유 ID를 지정합니다. (값은 HTML문서 내에서 고유해야 합니다.)

id 값은 CSS 및 JavaScript에서 지정된 id 값을 갖는 고유 요소에 대한 특정 작업을 수행한느 데 사용할 수 있습니다.

CSS에서 특정 id를 가진 요소를 선택하려면 요소의 id 다음에 해시(#) 문자를 씁니다.

ex)

id를 "myHeader"로 사용하여 요소 스타일을 지정하려면 CSS를 사용하십시오.

My Header

Tip: id 속성은 모든 HTML 요소에서 사용할 수 있습니다.

Note: id 값은 대 / 소문자를 구분합니다.

Note: id 값은 하나 이상의 문자를 포함해야 하면 공백(공백, 탭 등)을 포함해선느 안됩니다.

클래스와 ID의 차이점

HTML 요소는 단일 요소에 속한 하나의 고유 ID 만 가질 수 있지만 클래스 이름은 여러 요소에서 사용 될 수 있습니다.

ex)

London

London is the capital of England.

Paris

Paris is the capital of France.

Tokyo

Tokyo is the capital of Japan.
