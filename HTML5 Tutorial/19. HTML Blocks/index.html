HTML 블록 및 인라인 요소

모든 HTML 요소는 요소 유형에 따라 기본 표시 값을 가집니다. 대부분의 요소에 대한 기본 표시 값은 블록 또는 인라인 입니다.

Block-level Elements

display의 default가 block 인 elements 들을 말함.

블록 수준 요소는 항상 새줄에서 시작하여 사용할 수 있는 전체 너비를 사용합니다.(가능한 왼쪽 및 오른 쪽으로 뻗어 있슴)

< div > element는 Bloxk-level Elements입니다.

인라인 요소

이것은 단락 안에 있는 인라인 < span > 요소입니다.

< div >요소

< div >요소는 종종 다른 HTML 요소의 컨테이너로 사용됩니다.

< div >요소에는 필수 속성이 없지만 스타일, 클래스 및 ID 는 공통적입니다.

CSS 와 함께 사용하면 < div > 요소를 사용하여 콘텐츠 블록을 스타일 지정할 수 있습니다.

ex)

London

London is the capital city of England. It is the most populous city in the United Kingdom, with a metropolitan area of over 13 million ingabitants.
