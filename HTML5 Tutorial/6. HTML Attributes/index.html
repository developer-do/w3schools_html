HTML 속성

  1. 모든 HTML 요소는 속성을 가질 수 있습니다.
  2. 속성은 요소에 대한 추가정보를 제공합니다.
  3. 속성은 항상 시작태그에 지정됩니다.
  4. 속성은 일반적으로 다음과 같이 이름 / 값 쌍으로 옵니다. ex> name="value"

title 속성

This is a paragraph

Tip : HTML5는 소문자 속성 사용을 권유함.

HTML5 표준은 소문자 속성 이름을 요구하지는 않지만

W3C는 HTML에서 소문자를 권장 하고 XHTML과 같은 더 엄격한 문서 유형에 대해서는 소문자를 요구합니다.

Tip : HTML5 표준은 속성값을 따옴표로 묶을 필요가 없다.

나쁜 예 : < a href=https://w3schools.com>

좋은 예 : < a href="https://w3schools.com">

'' or "" 따옴표

속성 값으 둘러싼 큰 따옴표는 HTML에서 가장 일반적이지만 작은 따옴표도 사용할 수 있다.

경우에 따라 속성 값 자체에 큰 따옴표가 들어 있으면 작은 따옴표를 사용해야 합니다.

< p title='John "ShotGun" Nelson'>

< p title="John 'ShotGun' Nelson">

단원 요약

  • 모든 HTML 요소는 속성을 가질 수 있습니다.
  • 이 title 속성은 추가 " 도구 팁 " 정보를 제공합니다.
  • href 속성은 링크에 대한 주소 정보를 제공합니다.
  • width 및 height 속성은 이미지의 크기 정보를 제공
  • 이 alt 특성은 화면 판독기에 텍스트를 제공합니다.
  • W3Schools에서는 항상 소문자 속성 이름을 사용합니다.
  • W3 스쿨에서 우리는 항상 따음표 큰 따옴표 와 속성값을 사용한다.
