HTML Tables

HTML 테이블 정의하기

HTML 테이블은 < table > 태그로 정의됩니다.

각 테이블 행은 < tr >태그로 정의됩니다. 테이블 헤더가 < th >로 정의됩니다. 기본적으로 테이블 표제는 굵게 표시되고 중앙에 배치됩니다. 테이블 데이터 / 셀이 < td >태그로 정의됩니다.

ex)

Firstname Lastname Age
Jill Smith 50
Eve Jackson 94

Tip: < td > 요소는 테이블 데이터 컨테이너. 여기에는 모든 종류의 HTML 요소가 포함될 수 있습니다. 텍스트, 이미지, 목록, 다른 테이블 등

HTML 표 - 테두리 추가

표에 테두리를 지정하지 않으면 테두리 없이 테이블이 표시됩니다.

테두리는 CSS border 속성을 사용하여 설정 됩니다.

ex)

    table, th, td {
      border: 1px solid black;
    }

Note: 표 셀과 표 셀의 테두리를 정의해야 합니다.

HTML 표 - 축소 된 테두리

테두리를 한 테두리로 축소하려면 CSS border-collapse 속성을 추가하십시오.

ex)

    table, th, td {
      border: 1px solid black;
      border-collapse : collapse;
    }

HTML 표 - 셀 채우기 추가

셀 패딩은 셀 내용과 경계 사이의 간격을 지정합니다.

패딩을 지정하지 않으면 패딩없이 표 셀이 표시됩니다.

패딩을 설정하러면 CSS padding 속성을 사용하십시오.

ex)

    th, td {
      padding: 15px;
    }

HTML 표 - 왼쪽 맞춤 표제

기본적으로 테이블 표제는 굵게 표시되고 중앙에 배치됩니다.

표제를 왼쪽 정렬 하려면 CSS text-align 속성을 사용하십시오.

ex)

    th {
      text-align: left;
    }

HTML 표 - 테두리 간격 추가

테두리 간격은 셀 사이의 간격을 지정합니다.

표의 테두리 간격을 설정하려면 CSS border-spacing 속성을 사용하십시오.

ex)

    table {
      border-spacing : 5px;
    }

Tip: 테이블에 테두리가 축소되어 있으면 테두리 간격이 적용되지 않습니다. (collapse 를 정의해놨다면)

HTML 표 - 여러 열을 포함하는 셀

셀이 둘 이상의 열에 스팬되도록 하려면 다음 colspan 속성을 사용하십시오.

ex)

Name Telephone
Bill Gates 55577854 55577855

HTML표 - 많은 행을 포함하는 셀

셀이 둘 이상의 행에 걸쳐 있도록 하려면 다음 rowspan속성을 사용하십시오.

ex)

Name: Bill Gates
Telephone: 55577854
55577855

HTML 표 - 캡션 추가하기

캡션을 표에 추가하려면 < caption >태그를 사용하십시오.

ex)

Monthly savings
Month Savings
January $100
February $50

Tip: < caption >태그는 < table >태그 바로 뒤에 삽입해야 합니다.

하나의 테이블을 위한 특별한 스타일

특수 테이블에 대한 특수스타일을 정의하려면 테이블에 id 속성을 추가하십시오.

ex)

Firstname Lastname Age
Eve Jackson 94

이제 이 표에 대한 특수한 스타일을 정의 할 수 있습니다.

    table#t01 {
      width: 100%;
      background-color: #f1f1c1;
    }

그리고 더 많은 스타일을 추가하십시오.

    table#t01 tr:nth-child(even) {
      background-color: #eee;
    }
    table#t01 tr:nth-child(odd) {
      background-color: #fff;
    }
    table#t01 th {
      color: white;
      background-color: black;
    }

단원 요약

  • HTML < table > 요소를 사용하여 표 정의
  • HTML < tr > 요소를 사용하여 테이블 행 정의
  • HTML < td > 요소를 사용하여 테이블 데이터 정의
  • HTML < th > 요소를 사용하여 표제를 정의
  • HTML < caption > 요소를 사용하여 표 캡션 정의
  • border 테두리를 정의 하려면 CSS 속성을 사용해야한다.
  • CSS border-collapse 속성을 사용하여 셀 테두리 축소
  • CSS padding 속성을 사용하여 셀에 패딩 추가
  • CSS text-align 속성을 사용하여 셀 텍스트 맞춤
  • CSS border-spacing 속성을 사용하여 셀 사이의 간격 설정
  • colspan 속성으로 여러 컬럼으로 확장 할 수 있다
  • rowspan 속성으로 여러 행으로 확잘 할 수 있다.
  • id 특성을 사용하여 하나의 테이블을 고유하게 정의 할 수 있다.
