키보드 입력을 위한 HTML < kdb >

HTML < kdb > 요소는 키보드 입력 또는 음성 명령과 같은 사용자 입력을 나타냅니다.

< kdb >태그로 둘러싸인 텍스트는 일반적으로 브라우저의 기본 고정 폭 글꼴로 표시됩니다.

ex)

Save the document by pressing Ctrl + s

프로그램 출력을 위한 HTML < samp >

HTML < samp > 요소는 프로그램 또는 컴퓨터 시스템의 출력을 나타냅니다.

< samp > 태그로 둘러싸인 텍스트는 일반적으로 브라우저의 기본 고정 폭 글꼴로 표시됩니댜.

ex)

If you input wrong value, the program will return Error!

컴퓨터 코드용 HTML < code >

HTML < code > 요소는 컴퓨터 코드 조각을 정의합니다.

< code > 태그로 둘러싸인 텍스트는 일반적으로 브라우저의 기본 고정 폭 글꼴로 표시됩니다.

ex)

x = 5; y = 6; z = x + y;

< code > 요소에는 추가 공백 및 줄 바꿈이 보존되지 않는다는점에 유의 하십시오.

이 문제를 해결하려면 < code >요소를 < pre >요소에 넣으십시오.

ex)

    
      x = 5;
      y = 6;
      z = x + y;

변수는 수학 식의 변수 또는 프로그래밍 컨텍스트의 변수일 수 있다.
