HTML 이미지 구문

HTML에서는 이미지가 < img > 태그로 정의됩니다.

< img > 태그는 단지 속성을 포함, 비어있는, 그리고 닫는 태그가 없습니다.

src 소성은 이미지의 URL (웹 주소)를 지정합니다.

ex)

< img src="URL" alt="" >

alt 속성

alt 속성은 어떤 이유로 사용자가 볼 수 없는 경우, 이미지에 대한 대체 텍스트를 제공합니다. ( 떄문에 느린 연결로, src 속성의 오류, 또는 사용자가 화면 판독기를 사용하는 경우)

alt 속성의 값은 이미지를 설명합니다.

이미지 크기 - 너비와 높이

이 style 속성을 사용하여 이미지의 너비와 높이를 지정할 수 있습니다.

ex)

< img src="img_firl.jpg" alt="Firl in a jacket" style="width: 500px; height: 600px;" >은 아래와 같다.

< img src="img_girl.jpg" alt="Girl in a jacket" width="500" height="600" >은 아래와 같다.

width 및 height 속성은 항상 픽셀 이미지의 폭과 높이를 정의합니다.

Tip: 항상 이미지의 너비와 높이를 지정하는게 좋습니다. 너비와 높이를 지정하지 않으면 이미지가 로드되는 동안 페이지가 깜박일 수 있습니다.

너비와 높이, 또는 스타일?

width, height 및 style 속성은 HTML5에서 유효합니다.

그러나 이 style 속성을 사용하는 것이 좋습니다. 스타일 시트가 이미지 크기를 변경하지 못하게 합니다.

internal 방식으로 (내부 CSS) inline CSS 방식을 이길수 없다. 단 !important를 사용하면 된다.

이미지 map

map 태그는 이미지 맵을 정의합니다. 이미지 맵은 클릭 가능한 영역이 있는 이미지 입니다.

아래 이미지에서 컴퓨터, 전화 또는 커피 잔을 클릭하십시오.

태그의 name 속성은 usemap 속성과 연관되어 있으며 이미지와 맵 사이의 관계를 만듭니다. < map > <==> < img >

< map > 요소는 이미지 맵에서 클릭 가능한 영역을 정의하는 여러 < area > 태그를 포함합니다.

< picture > 요소

HTML5는 < picture >이미지 리소스를 지정할 때 더 많은 유연성을 추가 하는 요소를 도입했습니다.

이 < picture > element는 서로 다른 이미지 소스를 참조하는 < source > 요소를 포함한다. 이렇게 하면 브라우저가 현재보기 및 또는장치에 가장 잘 맞는 이미지를 선택할 수 있습니다.

각 < source >요소에는 이미지가 가장 적합한 시기를설명하는 속성이 있습니다.

브라우저는 속성 값이 일치하는 첫 번째< source > element를 사용하고 다른 < source >element는 무시합니다.

ex)

브라우저 창이 650픽셀 이상일 때 / 브라우저 창이 465 이상일 때 / 브라우저 창이 465 미만일 때 갈린다.

Tip: 항상 < img > 요소를 < picture > 요소의 마지막 자식 요소로 지정하십시오. < img > 요소는 < picture > 요소를 지원하지 않는 브라우저 또는 < source > 태그가 일치하지 않는 브라우저에서 사용됩니다.

HTML 스크린 리더

화면 판독기는 HTML 코드를 읽고 텍스트를 변환하며 사용자가 내용을 "들을" 수 있는 소프트웨어 프로그램 입니다. 스크린 리더는 시각 장애인이나 학습 장애자에게 유용합니다.

단원 요약

  • HTML < img >요소를 사용하여 이미지 정의
  • HTML src 속성을 사용하여 이미지의 URL을 정의
  • 이미지의 대체 텍스트를 표시하려면 HTML alt 속성을 사용하십시오.(이미지를 표시 할 수 없는 경우)
  • HTML width 및 height 속성을 사용하여 이미지 크기 정의
  • CSS width 및 height 속성을 사용하여 이미지 크기 정의
  • CSS float 속성을 사용하여 이미지가 뜨도록 할 수 있습니다.
  • HTML < amp >요소를 사용하여 이미지 맵 정의
  • HTML < area > 요소를 사용하여 이미지 맵에서 클릭 가능한 영역을 정의합니다.
  • HTML < img > 의 요소 usemap속성을 사용하여 이미지 맵을 가리킵니다.
  • HTML < picture > 요소를 사용하여 여러 장치에 서로 다른 이미지 표시

Tip: 이미지로드에는 시간이 걸립니다. 큰 이미지는 페이지 속도를 늦출 수 있습니다. 이미지를 신중하게 사용하십시오.
